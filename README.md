# Mood Prediction & Chatbot (FastAPI)

Project ini adalah aplikasi web sederhana untuk:
- Memprediksi mood berdasarkan kebiasaan harian
- Memberikan pendampingan ringan melalui chatbot

## Fitur
- Prediksi mood (happy / neutral / bad)
- Chatbot berbasis mood
- Web UI sederhana
- Backend FastAPI

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- HTML, CSS, JavaScript

## Cara Menjalankan
```bash
pip install -r requirements.txt
uvicorn main:app --reload
